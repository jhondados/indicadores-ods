# 🏛️ Dashboard de Indicadores ODS

> Painel de monitoramento dos Objetivos de Desenvolvimento Sustentável (ODS) por município com IA preditiva e benchmarking.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/indicadores-ods/

[![Vertical](https://img.shields.io/badge/Vertical-Governança-blue)]() [![IA](https://img.shields.io/badge/IA-ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**5.570** Municípios · **17** ODS · **248** Indicadores · **2030** Projeção

## 🧠 Tecnologias de IA
ML · Forecasting · Data Viz

## ✨ O que faz
- 17 ODS por município
- Projeção até 2030
- Benchmarking regional
- Priorização de políticas

## 🗄️ Fonte de dados real
[IBGE + IPEA](https://www.ibge.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com IBGE + IPEA
2. **Processamento com IA** — ML · Forecasting · Data Viz
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
